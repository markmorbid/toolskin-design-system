S1 — Live token values (theme-aware)

Hex values read from document.documentElement computed styles. Updates when you toggle light/dark.

Preview, contrast & surface export

Accent swatches set both the mini preview and the hex used by Apply accent to theme. Gradients change surface background only in the preview; contrast-safe text colors stay local to the preview card until you apply an accent swatch.

Global contrast tweak (surface maps)

Drag for a live preview on this page, then Apply contrast tweak to write --ts-text-secondary, --ts-text-muted, and a slight --ts-bg-2 adjustment into both dark and light theme maps (needs Color.js). If Color.js is missing, Apply only previews on the live page via color-mix. Clear contrast tweak reverts the last map apply or removes temporary page overrides.

Infinite scrolling text marquee with full CSS variable support, data attributes, and JavaScript API. Perfect for hero sections, tickers, and announcements.

Features

  • Configurable via CSS variables
  • Data attribute support (data-mq-*)
  • Pause on hover (optional)
  • Direction control (left/right)
  • Responsive and performant
  • Auto-calculates text repeats
  • Viewport-aware pausing
