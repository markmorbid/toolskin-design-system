Marquee Component

Infinite scrolling text marquee with full CSS variable support, data attributes, and JavaScript API. Perfect for hero sections, tickers, and announcements.

Infinite scrolling text marquee with full CSS variable support, data attributes, and JavaScript API. Perfect for hero sections, tickers, and announcements.

Features

  • Configurable via CSS variables
  • Data attribute support (data-mq-*)
  • Pause on hover (optional)
  • Direction control (left/right)
  • Responsive and performant
  • Auto-calculates text repeats
  • Viewport-aware pausing
