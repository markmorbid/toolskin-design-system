Global contrast tweak (surface maps)

Drag for a live preview on this page, then Apply contrast tweak to write --ts-text-secondary, --ts-text-muted, and a slight --ts-bg-2 adjustment into both dark and light theme maps (needs Color.js). §1b — Surface preset lab (testing)

Curated background and text tokens (delivered labels). Dark presets only update the dark-theme map; light presets only update the light-theme map — no cross-mixing.

Each card includes a description, a swatch ramp (surfaces + text), and code for that mode. Use Apply or activate the card (click / Enter / Space). Reset restores the factory defaults: Practical · Neutral (dark) and Practical · Clean neutral (light).
