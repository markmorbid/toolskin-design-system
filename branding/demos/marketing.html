§ Token engine

One accent, one stylesheet, every product.

Write once. Theme infinitely. Toolskin's tokens compose hierarchically — accent → ramp → component → page. Touch the accent, the system follows.

:root {
  --ts-accent-h: 18;
  --ts-accent-s: 100%;
  --ts-accent-l: 52%;
  /* derive everything */
  --ts-accent: hsl(var(--ts-accent-h),
    var(--ts-accent-s),
    var(--ts-accent-l));
  --ts-on-accent: oklch(from
    var(--ts-accent) clamp(...));
}
