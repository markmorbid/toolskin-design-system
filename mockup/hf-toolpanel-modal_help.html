Help & Instructions

Getting Started

The Higgsfield Automation panel allows you to queue and automate video generation on Higgsfield.ai. Import a CSV file or paste spreadsheet data to populate the queue.

CSV Format

Your CSV should contain columns: ID, image_URL, Prompt. The script will process each row sequentially.

Dashboard

Use the Dashboard to view queue progress, filter by status (Pending, Generating, Success, Failed), download completed videos, and manage individual items.

Pre-Fetch

The Pre-Fetch button scans the current page DOM for generated videos and queues them for bulk download, even if they weren't part of your original queue.

Controls

  • Start Queue — Begin processing all pending items
  • Pause — Pause processing (resume anytime)
  • Stop — Stop and reset processing state
  • Auto-download — Automatically download videos when generation completes

Logs

Toggle between Workflow Log (process messages) and Table Log (generation progress per item). Both can be copied, downloaded, or cleared.
