Help & Instructions

The main action buttons on this panel are enabled or disabled based on the content filter you have selected in YouTube Studio.

To use a specific function, first filter your videos on the 'Content' page:

  • Schedule / Rename: Requires the 'Visibility' filter to be set to Unlisted.
  • Un-Draft: Requires the 'Visibility' filter to be set to Draft.
  • Unlist: Requires the 'Visibility' filter to be set to Scheduled.
  • Rename Only: Requires the 'Visibility' filter to be set to Draft.
  • Disable All Notifications: Requires the 'Visibility' filter to be set to Unlisted.

If a button is disabled, hover over it to see which filter is required.

Auto-Publish Drafts

This tool automates the process of publishing multiple draft videos. It opens each draft, applies your chosen settings, and publishes it with the specified visibility.

Key Features

  • Bulk Publishing: Automatically process and publish all draft videos page by page.
  • Detailed Configuration: Set advanced options like Shorts remixing, content disclosures, category, and more for all videos in the batch.
  • Visibility Control: Choose to publish drafts as Public, Unlisted, Private, or Members-only.
  • Notification Control: Globally enable or disable the "Notify subscribers" option for the entire batch.

How to Use

  1. Filter Your Content: On the YouTube Studio "Content" page, filter your videos by "Drafts" visibility.
  2. Configure Settings: Adjust the options in this modal to your preference. These settings will be applied to every draft video processed.
  3. Start the Process: Click the "Confirm & Start" button to begin the automation.
  4. Monitor: The script will navigate through your drafts, apply the settings, and publish them. Keep the browser tab in focus for best results.
  5. Stop (If Needed): You can click the "Stop" button at any time to safely interrupt the process.
Disable All Notifications

This tool automates the bulk editing of the "Disable All Notifications" checkbox from un-published videos. It runs in two steps:

  • Fetch & Queue: It collects the edit page URLs of all videos and builds a processing queue.
  • Auto-Process: It navigates through each video’s edit page, unchecks the notification box, and saves the changes automatically.

Instructions:

  1. Filter Your Content: On the "Content" page, filter by "Unlisted" visibility.
  2. Click "Disable Notifs" and choose your processing options (e.g., process all, or use manual selection).
  3. If using Manual Selection, click "Start Selection Mode", choose your videos, and then "Confirm".
Transcript Downloader

This tool automates the fetching, downloading, and batch processing of YouTube video transcripts directly from the video page.

Key Features

  • One-Click Fetch: Automatically opens and retrieves the transcript for the current video.
  • Batch Processing: Enter a list of video or playlist URLs to download all their transcripts sequentially.
  • Auto-Fetch Mode: When enabled, the transcript is automatically fetched every time you load a new video page.
  • Persistent Settings: Your preferences for auto-fetching and including timestamps are saved automatically.

How to Use

  1. Manual Fetch: Click "Fetch Transcript" to get the transcript for the current video. "Download" and "Copy" will become enabled.
  2. Batch Mode: Click "Batch Process" to open the detailed control modal. Paste your URLs and click "Start Batch".
  3. Automatic Mode: Enable the "Auto-Fetch" checkbox. Transcripts will be fetched automatically on each video page you visit.
