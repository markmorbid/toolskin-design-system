Help & Instructions

Welcome to Suno Automation & Download

This panel provides powerful automation tools for Suno track generation and bulk downloading. All tools are accessible at all times and work together to streamline your workflow.

Getting Started

  • All tools are visible and accessible simultaneously — no need to switch between modes.
  • Use the progress card to monitor active processes in real-time.
  • The queue manager allows you to view, filter, and manage all fetched tracks before downloading.
  • Settings can be accessed from the panel header to customize behavior and appearance.

Tool Overview

Below you will find detailed instructions for each tool. Click on any section to expand and view detailed information.

Track Generation

Automate track creation on Suno. Navigate to suno.com/create and the panel will appear automatically.

  • Start Generating — Begins automated track creation using Suno's "Create" button. You can set limits for clicks and downloads.
  • Track Status — Opens a modal showing all generated tracks and their current status (completed, in-progress, failed).
  • Stop Clicks — Immediately stops the automation process.
  • Pause — Temporarily halts automation. Click again to resume.

Tip: Fill in the prompt textarea before starting. The automation will use whatever text is in the create form.

Bulk Downloader

Download all your Suno tracks in bulk. The downloader scans your workspace and collects track URLs for batch downloading.

  • Fetch All Tracks — Scans your Suno workspace and collects all available tracks. Progress is shown in the log area.
  • View Queue — Opens the download queue manager with search, filter, select, and batch actions.
  • Bulk Download All — Downloads all fetched tracks at once using your configured settings.
  • Copy — Copy the fetched track URLs to clipboard.
  • Save .txt — Export track list to a text file.

Note: Downloads use the format and filename pattern configured in Settings.

Queue Manager

The queue manager provides a detailed table view of all fetched tracks with powerful management tools.

  • Search & Filter — Find specific tracks by title, ID, or status.
  • Select / Deselect — Use checkboxes to select tracks for batch operations.
  • Download Selected — Download only the selected tracks.
  • Import/Export — Import existing files for deduplication or export queue data.
Settings

Access settings via the gear icon in the panel header.

  • Gradient Buttons — Toggle gradient styling on buttons.
  • Optimize UI — Disable transitions and animations for better performance.
  • Disable Backdrop Blur — Reduce GPU usage by removing blur effects.
  • Show Tooltips — Toggle hover tooltips on all interactive elements.
  • Layout Version — Switch between layout V1 and V2 (with dockable progress card).
  • Clear History — Reset automation, download, or all history data.
Logs

The log area at the bottom of the panel shows real-time progress and status messages.

  • Copy Log — Copy the entire log output to clipboard.
  • Download Log — Save the log as a text file.
  • Clear Log — Clear all log entries.
