ts-gallery · first draft

Asymmetric image grid built on the float-and-percentage method — authored layouts (quarter / half / two-third / full / double-height) instead of auto-flow flex. Sibling to .ts-masonry; not a replacement. Lazy-loaded, skeleton-shimmered, lightbox-ready, token-driven end to end.

Portfolio layout asymmetric · float-based

Mirrors the original CSS-only portfolio grid. Each <li> carries size classes; JS only hydrates lazy-loading + lightbox. Resize the window — the gallery rebreaks via container queries.

Uniform grid programmatic · sortable-friendly

Built from a JS array via ToolskinUIKit.Gallery.create(). All tiles are square; works smoothly with sortable mode (FLIP-friendly).

Sortable gallery drag to reorder

Adding data-ts-gallery-sortable auto-wires the existing ToolskinUIKit.Sortable engine. Pair with the uniform variant for smooth FLIP reordering — float layouts don't reflow cleanly in 2D under drag.
